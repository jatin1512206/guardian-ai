# Architecture Overview

## Component Breakdown
1. **Driver Behavior Analysis Agent**: Monitors eyes aspect ratios, facial attention vectors, and cellular phone distractions.
2. **Vehicle Dynamics & Road Context Agent**: Aggregates IMU accelerations, speed indexes, steering inputs, and lane drifts.
3. **Accident Prediction Intelligence Agent**: Performs temporal multi-modal sensor fusion via a Transformer Network.
4. **Intervention & Emergency Response Agent**: Issues warnings, vibration markers, automatic braking steps, and SMS notifications.
